# Quarto Portfolio Website Template (DA Colloquium)

This repository is a starter template for a simple, professional portfolio website built with **Quarto**.

You will:
1) Make your own copy (fork or use template)
2) Edit the pages (`index.qmd`, `projects.qmd`, `about.qmd`)
3) Preview locally with Quarto
4) (Optional) Publish to GitHub Pages

---

## 1) Make your own copy

### Option A: Fork (most common)
- Click **Fork** (top right).
- **Rename your repository during the fork** (recommended), e.g. `yourname-portfolio`.

### Option B: Use this template (if enabled)
- Click **Use this template** → **Create a new repository**
- Name it `yourname-portfolio`

---

## 2) Clone to your computer and open in RStudio

In RStudio:
- File → New Project → Version Control → Git
- Paste your repository URL
- Create project

---

## 3) Preview your website locally

In the Terminal pane (or system terminal), run:

```bash
quarto preview
